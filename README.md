Market Data Pipeline & Dashboard

Entwicklung einer End-to-End-Datenpipeline zur Analyse von Finanzmarktdaten unter Einsatz von Python, APIs und PostgreSQL.

• Aufbau einer automatisierten ETL-Datenpipeline zur Integration mehrerer Finanz-APIs 
• Konzeption und Implementierung einer PostgreSQL-Datenbank für strukturierte Datenspeicherung 
• Entwicklung interaktiver Power BI Dashboards (RSI, EMA, SMA, Fundamentaldaten) 
• Umsetzung von Drill-Down-Analysen für Aktien (AAPL, MSFT, GOOGL)
